## 新增 API 端点规划（前端调用）

### 认证相关

| 方法 | 路径 | 描述 | 前端页面 |
|------|------|------|---------|
| POST | `/api/auth/login` | 密码登录 | 登录页 |
| POST | `/api/auth/logout` | 登出 | 顶栏 |
| GET | `/api/app/profile` | 获取当前用户信息 | 布局框架 |
| GET | `/api/app/profile/security` | 获取账号安全信息 | 账号安全页 |

### OAuth 相关

| 方法 | 路径 | 描述 | 前端页面 |
|------|------|------|---------|
| GET | `/api/public/oauth/{provider}/authorize` | 生成授权 URL | 登录页 |
| GET | `/api/public/oauth/{provider}/callback` | OAuth 回调 | 登录页 |
| POST | `/api/app/auth/oauth/{provider}/bind` | 绑定三方账号 | 账号安全页 |
| DELETE | `/api/app/auth/oauth/{provider}/unbind` | 解绑三方账号 | 账号安全页 |
